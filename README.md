# Mini-project-2
This project is based on automated building detections 

automated-building-change-detection
Automated Building Detection and Change Analysis from satellite images using U-Net deep learning with Streamlit dashboard visualization.

Dataset
The project uses satellite building datasets such as LEVIR-CD. Due to large file size, the dataset is not included in this repository.

Dataset can be downloaded from: https://justchenhao.github.io/LEVIR/
